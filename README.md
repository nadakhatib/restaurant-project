# 🍽️ Sidra Restaurant

A simple restaurant website with beautiful views, food menu, and reservation system.

## 🔗 Live Demo
https://nadakhatib.github.io/restaurant-project/

## 📁 Pages
- **Home** – Restaurant views, gallery, seating
- **Menu** – Food categories with shopping cart
- **Contact** – Reservation form & location

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript

## ✨ Features
- Responsive design
- Shopping cart with localStorage
- Category filters
- Mobile menu

## 📍 Location
Gaza, Palestine – Al Sarya
